ABOUT

In a world that is continually progressing digitally and at an exponential rate, the use and understanding of technology is expected. For awhile, screens have been getting smaller and smaller, in which designers must figure out a way to display information so that it is easy for the user to comprehend and read -- a huge part of this is affected by the typefaces used on the interface.

LORETTA SERIF addresses these issuses that those with visual impairments struggle with using a larger x-height, slightly wider proportions, as well as lengthening ascenders and descenders. The typefaces that inspired this typeface were LORA and RETINA.
